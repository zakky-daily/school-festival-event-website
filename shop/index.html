出店紹介
『出店であなたを、驚かせたい』
文化祭はクラスや部活だけのものだと思ってませんか？増穂祭ではクラスや部活としてだけでなく、委員会や有志の学生、なんと先生まで！様々な団体が活躍できる「出店」という文化があります。
出店場所はグラウンドを中心に、校内の様々な場所に分布しています。ぜひ、増穂祭のツアーのように各々の出店を回ってみてください。ほかの文化祭では味わえない新鮮な体験があなたを待っています！

家庭科部
カップケーキ
# 飲食
家庭科部は校舎４Ｆの階段のそばで、手作りのカップケーキを販売しています♪
プレーン、チョコレートに、カラースプレーが乗ったカップケーキや体育の先生方とコラボしたプロテインカップケーキまで、様々な種類がありますので是非是非食べに来てください♪
バスケットボール部
パンチングマシーン
# アクティビティ
文化祭でどこを回ればいいか分からないゴリゴリの体育会系諸君！もう安心したまえ。我々バスケ部はそんな諸君らの為に体を動かすアクティビティとして、出店として思い切り殴れるサンドバッグを用意した！
もちろん利用は無料である！思うように動けなくてストレスが溜まっている奴らは、我らがバスケ部が出店するパンチングマシンに足を運んでくれたまえ！
サッカー部
キッキングクラブ
# アクティビティ
体育の授業でサッカーをやるけど上手く蹴れずに困ってる……そう思ってませんか？それなら私たちサッカー部が出店している「キッカー」に是非来てください！絶対に動かないサンドバッグを相手に、部員が個別で蹴り方のコツをお教えします。
また、ストレス発散が目的の方も大歓迎です！ぜひ来てください！！

体育教員
プロテイン販売
# ショップ
皆さん、筋肉足りてますか？足りてない？それなら今すぐにたんぱく質を補給しなければ！
